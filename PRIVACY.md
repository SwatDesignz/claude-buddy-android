# Privacy Policy — ZXBuddy™

**Last updated:** June 4, 2026

## Data Collection
ZXBuddy does **not** collect, store, or transmit any personal data. The app operates **offline-first** — all pet data, achievements, and game progress are stored locally on your device using Android Room (SQLite).

## Internet Access
- **AI Chat Features**: If you configure a Gemini or OpenRouter API key, the app sends your chat messages to those third-party APIs. No other data is transmitted.
- **No Telemetry**: The app contains no analytics SDKs, crash reporting, or usage tracking.

## Local Storage
The following data is stored **only on your device**:
- Pet profiles (name, species, level, stats)
- Achievement progress
- Game scores and ZX Points
- App logs and terminal history

This data can be deleted at any time by clearing the app's data or uninstalling.

## Third-Party Services
ZXBuddy optionally connects to:
- **Google Gemini API** — only if you provide an API key
- **OpenRouter API** — only if you provide an API key

Refer to their respective privacy policies for how they handle data.
